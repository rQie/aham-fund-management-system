## How to run

### Backend
1. Install dependencies

```bash
  cd backend yarn
```

2. Run the app

```bash
  yarn start
```

### aham-fund-management-system
1. Install dependencies

```bash
  cd aham-fund-management-system yarn
```

2. Run the app

```bash
  yarn dev
```
